# Supply Chain & Analytics Portfolio

This repository contains SQL and data analytics problem-solving examples focused on
supply chain, logistics, and operations use cases.

## Skills Demonstrated
- SQL (data filtering, aggregation, business queries)
- Analytical thinking
- Translating data into business insights
- Operations and supply chain reasoning

## Contents
- SQL practice problems with business explanations
- Real-world analytics case-style problems 

This portfolio is intended to demonstrate practical problem-solving ability rather
than just technical syntax.
